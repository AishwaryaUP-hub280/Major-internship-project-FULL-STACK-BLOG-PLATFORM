# Full Stack Blog Platform

## 📌 Project Description
This project is a full stack blog platform where users can:
- Register and Login
- Create, Read, Update, Delete blog posts

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- API Testing: Postman

## 🚀 Features
- JWT Authentication
- CRUD Operations on Blogs
- Secure API Routes

## ▶️ How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start  

## 🌐 URLs
- Frontend: http://localhost:3000  
- Backend: http://localhost:5000
